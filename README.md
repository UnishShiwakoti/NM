# Numerical Methods Lab Work

A collection of Python implementations for numerical methods algorithms studied in college coursework.

## 📋 Overview

This repository contains solutions and implementations for various numerical methods commonly used in mathematics and scientific computing. Each lab covers fundamental algorithms and techniques for solving mathematical problems numerically.

## 📚 Lab Topics

Add your specific lab topics here. Common topics include:

- **Root Finding Methods**
  - Bisection Method
  - Newton-Raphson Method
  - Secant Method

- **Linear Equations**
  - Gaussian Elimination
  - LU Decomposition
  - Jacobi Iteration

- **Interpolation**
  - Lagrange Interpolation
  - Newton's Divided Differences

- **Numerical Integration**
  - Trapezoidal Rule
  - Simpson's Rule

- **Differential Equations**
  - Euler Method
  - Runge-Kutta Methods

## 🛠️ Requirements

- Python 3.x
- NumPy (if used)
- Matplotlib (for plotting results)
- Pandas
